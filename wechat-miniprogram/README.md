# 问道阳明，竹海修心 - 微信小程序

## 项目简介

"问道阳明，竹海修心"是一款专为河源市和平县阳明博物馆开发的微信小程序，旨在为游客提供便捷的博物馆导览、文创购买、体验预约等服务。小程序采用中国传统风格设计，融入竹文化元素，打造沉浸式的王阳明文化体验平台。

## 功能模块

### 1. 问道阳明
- 博物馆导览：包含博物馆简介、开放时间、参观路线推荐、阳明文化核心脉络解读
- 场馆导航：提供停车场、卫生间、休息区、咨询台、充电桩等点位导航
- 语音讲解：支持全展厅和重点文物的真人语音讲解，一键开启/关闭

### 2. 阳明有礼
- 文创产品展示：全系列文创产品浏览
- 线上购买：支持快递配送和馆内自提两种方式
- 组合套餐：提供优惠的产品组合套餐

### 3. 体验预约
- 剧本杀预约：王阳明主题剧本杀体验
- 研学课程：竹文化研学课程预约
- 竹宴体验：特色竹乡美食体验预约
- 实时显示可预约时段

### 4. 我的
- 订单管理：查看文创购买订单
- 预约管理：查看体验活动预约
- 个人中心：收藏、足迹、积分等功能

## 技术实现

- 前端：HTML5 + CSS3 + JavaScript
- UI框架：Tailwind CSS v3
- 图标库：Font Awesome
- 响应式设计：适配不同屏幕尺寸的移动设备

## 使用说明

1. 打开微信，扫描小程序二维码
2. 进入首页，浏览博物馆信息
3. 点击底部导航栏切换不同功能模块
4. 在"问道阳明"模块了解博物馆信息和进行导览
5. 在"阳明有礼"模块浏览和购买文创产品
6. 在"体验预约"模块预约各类体验活动
7. 在"我的"模块管理订单和预约

## 设计特色

- 中国传统风格与现代UI设计相结合
- 竹文化元素贯穿整个界面
- 深绿色、浅绿色、棕褐色为主的配色方案，体现自然与文化的融合
- 简洁明了的导航设计，方便游客快速找到所需功能

## 注意事项

- 本项目为前端展示原型，实际使用需要对接后端API
- 部分功能（如在线支付、用户登录等）需要微信小程序的相关权限
- 语音讲解功能需要实际的音频资源支持
